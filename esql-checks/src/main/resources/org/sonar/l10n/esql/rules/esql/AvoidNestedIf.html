This rule applies whenever the nested if statement is used .

It is good programming practice to avoid nested IF statements: use ELSEIF or CASE WHEN clauses to get quicker drop-out.

Noncompliant Code Example

	IF FALSE THEN
			IF FALSE THEN
				IF FALSE THEN
					IF FALSE THEN 
						SET OutputRoot=InputRoot;
					END IF;

Compliant Solution

	ELSEIF A = B THEN 
	  ELSEIF FALSE THEN 
		 ELSEIF end THEN
