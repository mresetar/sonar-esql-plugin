The filter node cannot modify the message.

Noncompliant Code Example

	CREATE FILTER MODULE Flow2_Filter
      CREATE FUNCTION Main() RETURNS BOOLEAN
      BEGIN
        SET OutputRoot.XMLNSC.Response.details.person.age = '20';
       RETURN FALSE;
      END;
     END MODULE;

Compliant Solution

	CREATE FILTER MODULE Flow2_Filter
      CREATE FUNCTION Main() RETURNS BOOLEAN
      BEGIN
         RETURN FALSE;
      END;
     END MODULE;
