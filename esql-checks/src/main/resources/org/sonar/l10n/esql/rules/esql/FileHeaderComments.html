Each ESQL file should contains header information as below:

Noncompliant Code Example

	/*
	Change Log:
	S.No. Incident Number  Jira Number  Description  Date Fixed  Updated 
	
	*/

Compliant Solution

	/*
	Change Log:
	S.No. Incident Number  Jira Number  Description  Date Fixed  Updated ESQL/Node
	1       INC0182018        DI-999       Test defect  09/01/2017  test.esql
	*/
