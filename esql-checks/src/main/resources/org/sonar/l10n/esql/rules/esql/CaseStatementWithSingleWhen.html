This rule applies whenever the CASE Statement having single when in the esql, it should be replaced by IF Statement.

Replace this "CASE" statement by "if" statements to increase the code readeablity.

Noncompliant Code Example

	CASE size
		  WHEN minimum + 0 THEN
		    SET description = 'small';
		    END CASE;

Compliant Solution

	CASE size
		  WHEN minimum + 0 THEN
		    SET description = 'small';
		  WHEN minimum + 1 THEN
		    SET description = 'medium';        
		 END CASE;
